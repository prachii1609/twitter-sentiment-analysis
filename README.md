# Twitter Sentiment Analysis

## Description
This project analyzes tweets and classifies them into Positive, Negative, or Neutral using Machine Learning.

## Features
- Text-based sentiment prediction
- ML model using Naive Bayes
- Web app using Streamlit

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## How to Run
1. Install dependencies
2. Run:
   streamlit run app.py
